Jackie's RPG Maker

RPG Maker Clone...literally figuratively



Github: Jackie's RPG Maker

Developer: Jacqueline

C#, SFML, Dock Panel Suite

Oh boy, one of my first massive project I ever created. Ever wanted to make your own game? Yeah this is it. The system was designed to be OOP heavy, and I mean it, OOP HEAVY. NChatN

Chat a smugly as you feel



Github: NChatN

Offical page: NChatN Main page

Developer: Kevin

Javascript

Due to the deficiency of the original Nowhere-Else and Beyond chat client, especially in the wake of poor internet connections I took it upon myself to write something that could properly handle such errors without being an inconvience to the people using the client. As I worked I also added features that I personally wanted the client to have, such as input history, proper tab completion of usernames, the ability to preview an image by hovering over the link, and one of the most lauded features moving private conversations with players into their own dedicated tab instead of inline with general chat. It's hard to describe all the features and fixes NChatN had to those who never had the pleasure of using the original Nowhere-Else and Beyond chat client. Suffice it to say that NChatN was a significant improvemnt to the chatting experience of the game. So much so that the client was eventually officially moved into the game itself.

Implemented Features:

  • View some of what's been said in chat before you entered
  • Better username Tab-completion (can complete usernames in the middle of a sentence)
  • Keeps a full record of what's been said in chat since you opened the window
  • Allows you to easily save all of what's been said to your computer
  • Better scrolling support (no longer jerks down when someone says something in chat, and keeps your position when resizing the window)
  • Visual indicator shows when scrolling up into chat history
  • Ability to have more than four chat channels open at once (not that anyone needs that ability with NEaB's chat)

Plushie (Teh) Bot

The chat bot that hugs back



Github: PlushieBot

Developer: Kevin

Python

An attempt at creating a chat bot for Nowhere-Else and Beyond that evolved into more than just an experiment. Plushie was incrementally improved from an initial test to see how easy it was to create a chat bot, to a simple IRC-type bot, and finally to a more complex plugin-style bot. Plushie started taking on tasks that the resident chat bot couldn't handle correctly, such as playing games with players and doing math that included more than the standard addition, subtraction, multiplication, and division. Initially development was kept private, but due to the simplicity of the plugins an players request for small changes I decided to open source him and allow other players to contribute to him. Unfortunately Nowhere-Else and Beyond has died, but part of Plushie will eventually be pulled into another chat bot I make (and he can still be run on his own using the testing console I built for him).

Implemented Features:

  • !command style interaction
  • Mafia/Werewolf game manager
  • Hangman
  • Chat history querying
  • Wikipedia searches with page summary
  • Calculator support (could do all basic operations, sin, cos, tan, pi, factorials, and a few others)
  • Various small commands like random number generation and picking between a list of options
