Jackie's RPG Maker

RPG Maker Clone...literally figuratively



Github: Jackie's RPG Maker

Developer: Jacqueline

C#, SFML, Dock Panel Suite

Oh boy, one of my first massive project I ever created. Ever wanted to make your own game? Yeah this is it. The system was designed to be OOP heavy, and I mean it, OOP HEAVY. The whole system is made of base objects that was setted up to be polymorphed into other objects, which allow you to easily have database of similar but a bit different objects for things like Events, different functionals characters or buttons. It was a fun experience to create this, though if I came back to remake this, I will definitely aim for a better GUI and more stable foundation

Implemented Features:

  • Create characters that are playable (though you can only play the first and there is no switching)
  • Map, Map Editor, Map Animation
  • Event System like RPG Maker
  • Create/Save/Open game files.

Smite Team Management

Allow you to be a Smite professional... hopefully



Github: Team Smiter

Developer: Jacqueline

C#, SFML, Lidgren

A more professional project created recently. I made it for a Smite team coach in Switzerland. This software allow you to set up a local server where your team can join and draw strategies... literally. This is a whiteboard software that come with chat (decently private, by that I mean there is no tracking, but there is no encryption neither). That's pretty much it really, you can choose different maps, draw/erase, set the sizes, dragging gods around, etc. On a personal note, Awesomenaut > Smite.

Implemented Features:

  • Whiteboard, Draw/Erase over maps (and your friends can see it and draw with you, too)
  • Host your own server with friend, come with password for protection in case stranger join
  • Private chat, with whiteboard "privilege". Hoster can grant/take away whiteboard privilege
  • Pick, arrange your Smite team, drag them around strategically

Wandering Soul

An offline Haven and Hearth wannabe



Github: Wandering Soul

Developer: Jacqueline

C#, SFML

My most proud project I ever created, took me over 4 months to work on this. I was in love with a game called Haven and Hearth back then (still do, but I don't want to pay for the game now that a P2P/P2Win version came out). Definitely a project that I will come back later once I find the time. I spend a lot of time planning and designing this, though I was really disappointed that I was the only team member. The amount of work I had to do was overwhelming for a one person team. I did all the arts, all the designs, all the GUI, all the graphics, all the programming, etc. Though I think you should try it out, despite it's not finished. There are a lot of feature implemented (some of which I'm really is proud of).

Implemented Features:

  • Dynamic building setted up, you can make more building easily, but there is only one kind of building at the moment, and that's a bundle of wood (which you can actually lit it up)
  • Dynamic environment, again, only trees are cuttable at the moment, but with very ltitle coding, you can create more things to destroy
  • Autotile
  • Pathfinding
  • Basic AI
  • Map Generator
  • 2 handed combat, inventory/equipment, multiple characters/party system

Flashcard Quiz

Yes I know there are better one on the market



Github: Flashcard Quiz

Developer: Jacqueline

C#, WinForm, PDF Sharp

I made this during my SAT preps summer, because I didn't want to spend time studying the SAT. I do really well on math, but I just don't like doing English SAT stuff. Programming had more pull to me than SAT ever will do, so instead of creating flash card and memorize it, I just made a flashcard program instead. It was quite awesome actually, it took me like a week to code the program, and another weeks to add in vocabs words (manually) so that my friends can download it and print right away.

Implemented Features:

  • Word Database, add, remove your own words
  • Manage your lists, learn a few word at time, create smaller lists out of your big list
  • Print your words lists (make sure to use 2 sided printer)
  • Quiz yourself, with multiple choices or fill in the blank
